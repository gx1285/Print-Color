# How to use
Now vertion install
```sh
# Linux/macOS
python3 -m pip install -U color-printtext

# Windows
py -3 -m pip install -U color-printtext
```
Install with
Sample code
```py
import colorprinttext
colorprinttext.Redprint("TEST")
```
