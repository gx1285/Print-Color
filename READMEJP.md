printの色を指定するモジュール。
# 使い方
通常版
```sh
＃Linux/macOS
python3 -m pip install -U color-printtext

＃ Windows
py -3 -m pip install -U color-printtext
```
でインストールします。
サンプルコード
```py
import colorprinttext
colorprinttext.cprint('test','red')
```
